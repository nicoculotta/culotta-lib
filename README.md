# Pokedex coding challenge: Storybook library

## Installation:

- clone repo
- npm install

Run local storybook:

- npm run storybook

Build library:

- npm run build:library

Build storybook:

- npm run build-storybook

Technologies used:

- Storybook
- Rollup
- Jest
- Testing Library
- ESLint
- Prettier
- TypeScript
- React
- SCSS

Deployment:

- Github Actions
- NPM

To install the library in a project:

- npm install culotta-lib

Published version: https://www.npmjs.com/package/culotta-lib
